# Manual de Sobrevivência do 1º Ano Técnico - ETEC

Guia prático para calouros da ETEC com dicas de estudos, convivência, amizades e conselhos de veteranos.

## Capítulos

- **Capítulo 1** – Como passar em todas as matérias com notas boas
- **Capítulo 2** – Cuidados na ETEC (tempo, equipamentos, ambiente, respeito)
- **Capítulo 3** – Guia prático para iniciar conversas e fazer amizades
- **Capítulo 4** – Conselhos de um veterano para quem está chegando
